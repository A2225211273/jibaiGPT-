# 禾序 UI 设计说明

## 状态

第一版 UI 方向已于 2026-07-15 确认。本目录保存后续实现应遵循的正式视觉基线；界面仍使用演示数据，尚未连接 Codex `app-server`。

## 正式资源

- `hexu-ui-v1.png`：1536×1024 亮色完整展示板，也是第一版主稿
- `hexu-ui-dark-v1.png`：1536×1024 暗色完整展示板
- `hexu-ui-desktop-v1.png`：960×760 桌面悬浮状态
- `hexu-ui-attached-v1.png`：1240×760 Codex 窗口外侧贴附状态
- `hexu-ui-conversation-v1.png`：1240×620 会话内结构化用量卡片

这些文件由 `design/drafts/` 中确认通过的 `v2` 审阅稿原样归档。ImageGen 概念稿与未采用的早期截图继续留在草稿目录，不作为实现依据。

## 视图

- 完整稿：桌面主面板、贴附模式、会话卡片同屏展示
- 桌面：380×520 目标比例的悬浮面板
- 贴附：位于 Codex 窗口外侧的紧凑序列条
- 会话：通过插件工具返回的结构化结果，不宣称为官方原生侧边栏

## 文案规则

- 默认提醒保持中性。
- “松弛模式”默认关闭，必须由用户主动开启。
- 演示句：`你已经工作了 1小时，再忙也要导一管`。
- 所有用量数据均显示来源与更新时间；无法确认真实来源时不显示数字。
