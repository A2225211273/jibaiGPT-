# 禾序 UI 设计说明

## 状态

当前是第一版高保真原型，使用演示数据。正式截图需在用户确认视觉方向后保存到本目录；未确认的 ImageGen 构图稿放在 `design/drafts/`。

## 视图

- 完整稿：桌面主面板、贴附模式、会话卡片同屏展示
- 桌面：380×520 目标比例的悬浮面板
- 贴附：位于 Codex 窗口外侧的紧凑序列条
- 会话：通过插件工具返回的结构化结果，不宣称为官方原生侧边栏

## 文案规则

- 默认提醒保持中性。
- “松弛模式”默认关闭，必须由用户主动开启。
- 演示句：`你已经工作了 1小时，再忙也要导一管`。
- 所有用量数据均显示来源与更新时间；无法确认真实来源时不显示数字。
